BuSo Lightning is the fastest loading, cleanest theme you'll ever experience.

== Description ==

Made with page speed loading times in mind, BuSo Lightning was developed to be the cleanest most-minimal launching point for any website using the Wordpress content management system.  Deliver your users the content they demand faster than a bolt of lightning, and they'll stay on your site longer, click more ads, and purchase more products.  Start with the fastest theme and save yourself the headaches and regret later.

== Installation ==

1. Upload `busolightning.zip` to the `/wp-content/themes/` directory
2. Activate the theme through the 'Themes' menu in WordPress
3. Browse to the Appearance > Theme Options page.
4. Take note of the instructions at the top of the page, which explain the options available.
5. Enjoy!
